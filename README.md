# Embedded Mini Projects

Embedded mini projects using AVR/Arduino with register-level programming, Embedded C, and hardware interfacing.

## 1. 4-Digit Counter with User Input

A 4-digit multiplexed 7-segment counter system developed using AVR microcontroller and Embedded C.

### Features

* 4-digit multiplexed 7-segment display
* Digit selection using shift button
* Increment and decrement functionality
* Blinking cursor in edit mode
* Compare value setting
* Automatic counting mode
* Match detection with blinking display
* Register-level AVR programming

### Technologies Used

* Embedded C
* AVR Microcontroller
* GPIO Programming
* 7-Segment Display Interfacing
* Push Button Interfacing
